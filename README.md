# UEXO x Ektifaa Network Hierarchy

Vercel-ready Next.js prototype for Ektifaa IB/Sub-IB hierarchy management.

## Deploy on Vercel
1. Upload this folder to GitHub.
2. Go to Vercel > New Project.
3. Select this repository.
4. Click Deploy.

## Local run
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Important
This is a frontend prototype only. It does not yet save data or connect to CRM/MT5.
